Tipos de Requisições HTTP

Método Descrição
GET Busca dados do servidor.
POST Envia dados para criação.
PUT Atualiza totalmente um recurso.
PATCH Atualiza parcialmente um recurso.
DELETE Remove dados.
OPTIONS Consulta métodos permitidos.
HEAD Retorna apenas headers.

Exemplo fetch()

fetch('https://api.site.com', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    nome: 'Aitu'
  })
})
